Exercício 8.3

Operadores JavaScript - Parte 2

Os operadores em JavaScript são:

Aritméticos;
Atribuição;
Relacionais;
Lógico;
Ternário.

Operadores Lógicos

(!) Exclamação - "Não" - Operador lógico de Negação.
(&&) Ê comercial - "E" - Operador lógico de Conjunção.
(||) 2 Pipes - "OU" - Operador lógico de Disjunção.

Caso 1 - &&

true && true = true;
true && false = false;
false && true = false;
false && false = false;

Caso 2 - ||

true || true = true;
true || false = true;
false || true = true;
false || false = false;
